Awsome responsive look

We live in a multi-screen society. Because of this, it’s important for your site to be viewable across as many devices as possible, because you never know what device someone will be using to view your website.

Responsive design can help you solve a lot of problems for your website. It will make your site mobile-friendly, improve the way it looks on devices with both large and small screens, and increase the amount of time that visitors spend on your site. It can also help you improve your rankings in search engines.

About Me

Hi, my name is Mohamed Hany, UI, UX & front end web designer from Egypt. I love designing web sites and sometimes I love to get my hands dirty with logo design and pictures editing. I’m very good at writing semantic HTML5 markup, CSS & JavaScript code and I use libraries as Bootstrap & jQuery with some knowledge about Three.JS.

I’m a self-learner and I first learn to code at Udacity in 1 Million Arab Coders Initiative and complete two courses “Front End Web Development” and “Full Stack Web Development” then I took “Web Design for Everybody (Basics of Web Development and Coding) Specialization” at Coursera, offered by University of Michigan, which made me understand the importance of accessibility and wireframes and how to write clean code and check it. Now I’m locking to improve my skills in front end web design and learning back end web development.
